# Prerequisites Knowledge

Please find below a list of resources that I recommend to get familiar with the concepts and tools that we will be using in this course.

The goal is **not to fully master** all the concepts and tools in this list, but to get familiar with them and to be able to use them in practice.

You **do not need the physical platform** for these courses.
